Servidor streaming Video

Instalacion/Configuración

  • Instalamos el paquete de icecast con un sudo apt-get install icecast2. IMAGEN1
  • Damos a no, para configurarlo posteriormente en los ficheros de configuración. IMAGEN2
  • Accedemos al fichero /etc/icecast2/icecast.xml
    • Primero modificamos las contraseñas y un usuario para administrarlo. IMAGEN3
    • Cambiamos también el nombre (previamente añadido en el servidor DNS) y el puerto con el que nos vamos a conectar desde un cliente. IMAGEN4
  • Accedemos a /etc/default/icecast2 y ponemos ENABLE=true IMAGEN5
  • Instalamos el paquete ffmpeg2theora. IMAGEN6
  • Instalamos el paquete oggfwd. IMAGEN7

  • Tenemos que meter un video para reproducirlo, como es un Ubuntu Server tendremos que montar un USB para acceder al él, porque lo habíamos descargado previamente.

    • Lo montamos en un directorio previamente creado. IMAGEN8
    • Lo copiamos a la ubicación desde la que queramos ejecutarlo. IMAGEN9

  • Ejecutamos el comando para que empiece a reproducirse el video. IMAGEN10

Comprobaciones

  • Desde el cliente accedemos con el navegador, indicando el nombre de dominio del servidor y el puerto. IMAGEN11
  • Damos a Administración y nos logamos. IMAGEN12
  • Observamos la configuración. IMAGEN13
  • Desde el cliente observamos cómo se reproduce el video, que previamente hemos puesto a reproducir en el servidor.IMAGEN14
